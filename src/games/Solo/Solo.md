# Solo

v0.1a of Duo

## Game Play

### v0.1

A chord shows up on screen
  - Include paino chords, sheet music, and sax fingering
Metronome plays
  - Chord plays for 4 beats
  - Solo records for 4 measures
Audio input grabs signal, pitchfinds
  - For every frame the note is in key +1 point
  - For every new note you hit, + 50
  - For hitting every note in the chord + 20
